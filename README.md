# NexUI ⚡

**NexUI** is a modern, dark-themed React component library designed for developers who value both **beautiful design** and **development efficiency**.

It provides ready-to-use, reusable, and highly customizable UI components to help you build consistent and modern user interfaces faster.

---

## ✨ Features

- 🌙 Modern dark-themed UI design
- ⚛️ Built with React (functional components)
- 🎨 Glassmorphism-inspired styling
- ♻️ Fully reusable and modular components
- ⚡ Fast integration into any React project
- 🎛️ Customizable via simple props
- 📱 Responsive design for all screen sizes

---

## 📦 Components

### 🔝 Navbar
- Fixed top navigation bar
- Glass-like modern design
- Smooth scrolling between sections
- Active link highlighting
- Easy navigation across components

---

### 🔘 Buttons
- Multiple variants:
  - Primary
  - Secondary
  - Ghost
  - Danger
  - Success
- Supports different sizes
- Includes loading & disabled states
- Fully customizable via props

---

### 🃏 Cards
- Flexible layout system
- Suitable for:
  - User profiles
  - Dashboards
  - Content display
- Smooth hover animations
- Clean and modern UI structure

---

### 🪟 Modals
- Interactive popup components
- Types include:
  - Information modal
  - Confirmation modal
  - Warning modal
- Easy control with state management
- Smooth open/close transitions

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS Variables
- Modern UI/UX principles

NexUI focuses on:

Clean and minimal UI
Consistency across components
Developer-friendly structure
Modern dark aesthetic
Performance and reusability

TypeScript support

More advanced components (Tables, Dropdowns, etc.)
